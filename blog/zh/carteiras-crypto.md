## 什么是加密货币钱包？

加密货币钱包并不"存储"你的加密货币——它存储证明你拥有它们的私钥。把它想象成一个钥匙链：钥匙（私钥）打开门并允许你移动里面的东西，但保险箱本身在区块链上。

每个钱包生成一对密钥：
- **公钥：** 你的收款地址（就像银行账号）
- **私钥：** 你的发送主密码（切勿分享）

## 钱包类型

**托管钱包：** 第三方保管你的私钥。例如：WAX Cloud Wallet。你可以用邮箱和密码访问。公司持有密钥。

**自主保管：** 你控制自己的私钥。例如：Anchor Wallet。你拥有完全控制权——以及完全责任。

**热钱包 vs 冷钱包：** 热钱包连接互联网（适合游戏）。冷钱包离线（更适合大额存储）。

## WAX Cloud Wallet

推荐给初学者。像任何在线账户一样工作：邮箱 + 密码 + 验证。无需管理助记词。30秒即可创建账户。

**优点：** 简单，可通过邮箱恢复，与 WAX 生态集成
**缺点：** 你不控制私钥

## Anchor Wallet

完全自主保管。你生成并控制 12 个单词的助记词。支持多条 Antelope 区块链（WAX、EOS、Telos）。

**优点：** 完全控制，多链支持，开源
**缺点：** 你对助记词负责——丢失等于失去访问权限

## 选择哪一个？

| 标准 | WAX Cloud Wallet | Anchor Wallet |
|------|------------------|---------------|
| 易用性 | 优秀（30秒） | 中等（5分钟） |
| 控制权 | 托管 | 自主保管 |
| 助记词 | 不需要 | 12个单词 |
| 恢复方式 | 邮箱+密码 | 助记词 |
| 适合人群 | 初学者、游戏 | 高级用户 |

**我们的建议：** 从 WAX Cloud Wallet 开始。先玩起来，熟悉之后。当你拥有更多代币或想要更多控制权时，再迁移到 Anchor。

## 安全提示

- **助记词：** 切勿数字化。写在纸上并存放在安全的地方。实体保险箱更佳。
- **双重认证：** 尽可能启用双因素认证。
- **假网站：** 连接钱包前务必验证 URL。
- **切勿分享私钥：** 没有任何合法服务会索要你的助记词。
