## 安全是基础

在 CryptoBingo，安全不是一项功能——它是基础。作为一个链上 GameFi 平台，每一步都记录在公开、可审计且不可篡改的区块链上。没有"相信我们"——只有"自己验证"。

## 资金保护

**可审计的智能合约：** CryptoBingo 合约在 WAX 链上。任何人都可以阅读代码并验证规则——没有黑箱。

**连接钱包，非托管：** 你的资金留在你的钱包中（WAX Cloud Wallet 或 Anchor）。CryptoBingo 从不托管你的代币。你只授权特定交易（购买彩票、领取奖励）。

**自动支付：** 奖励由合约自动支付。无提款队列、无需人工审核、无"处理中"。如果你赢了，合约就会支付。

## 抽奖完整性

每次抽奖经过三层验证：

**1. 可验证公平：** 抽奖前，未来结果的哈希值记录在区块链上。抽奖后，你可以验证哈希值是否匹配结果——确保无人篡改。

**2. 混合RNG：** 随机数结合玩家承诺 + 服务器承诺 + Oracle RSA 签名 + nonce。任何单一角色都无法预测或操纵。

**3. 独立Oracle：** 随机数由外部 oracle（orng.wax）使用 RSA-SHA256 生成。连 CryptoBingo 也无法控制结果。

更多信息请阅读"可验证公平"和"RNG工作原理"。

## 玩家隐私

**无需强制KYC：** 连接钱包即可游戏。无需文件，无需自拍。

**假名制：** 你在区块链上的身份是你的 WAX 地址——一个字母数字哈希值。链上不存储任何个人数据。

**最少链下数据：** 仅存储非敏感元数据（偏好、比赛历史），MongoDB 中加密存储。

## 平台保护

**速率限制：** API 端点有请求限制以防止滥用。

**RBAC（访问控制）：** 管理操作使用细粒度权限（管理员、版主），全部记录在案。

**JWT密钥轮换：** 会话令牌过期并自动续期。

**空钱包清理：** 工作进程定期移除无余额账户，释放网络资源。

## 诚实风险

**合约风险：** 尽管代码公开且可审计，但没有智能合约是100%无错误的。我们通过测试持续改进。

**人为错误：** 你可能将代币发送到错误地址或丢失钱包访问权限。始终验证地址并保管好助记词。

**市场风险：** WAX 奖励的价值可能波动。美元价值不固定。

## 审计与透明度

所有合约操作都是公开的：
- 彩票购买
- 抽奖和结果
- 奖励支付
- 游戏配置
- 资金库变动

任何人都可以通过区块浏览器（如 WAX Block Explorer）进行监控。

CryptoBingo 的安全不是承诺——是代码。而代码你可以验证。
