## 测试网开放

CryptoBingo 现已在 WAX 区块链上正式进入测试网阶段。这是平台的第一个公开版本，允许社区在正式发布前测试所有功能。

### 可用功能

- 使用 WAX 购买彩票
- 具有不同开奖模式的游戏房间
- WAX 和 NFT 奖品
- 管理后台
- 可验证的 RNG 预言机

### 参与方式

1. 创建 WAX 钱包（Anchor 或 Cloud Wallet）
2. 获取测试 WAX 代币
3. 访问 test.cryptobingo.bet
4. 购买彩票并开始游戏！
