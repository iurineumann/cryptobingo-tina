## Testnet Open

CryptoBingo is now officially in testnet on the WAX blockchain. This is the first public version of the platform, allowing the community to test all features before the official launch.

### What's Available

- Buy tickets in WAX
- Game rooms with different draw modes
- Prizes in WAX and NFTs
- Administrative dashboard for management
- Verifiable RNG oracle

### How to Participate

1. Create a WAX wallet (Anchor or Cloud Wallet)
2. Get test WAX tokens
3. Visit test.cryptobingo.bet
4. Buy tickets and play!

### Next Steps

We are working on continuous improvements based on community feedback. Stay tuned on Telegram and Discord for updates.
