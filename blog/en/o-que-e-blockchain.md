## What is a Blockchain?

Imagine a notebook that everyone can see and no one can erase. Each new page (block) contains a set of transactions and a reference to the previous page. This notebook isn't kept in anyone's drawer — copies of it exist on thousands of computers around the world simultaneously.

That is a blockchain: a shared, decentralized, and immutable digital ledger.

## How It Works in Simple Terms

**Blocks:** each block is like a page in the notebook, containing transactions (ticket purchases, draws, prizes).

**Chain:** each block contains a hash (digital fingerprint) of the previous block. If someone tries to alter an old block, the hash changes and the chain breaks — everyone notices.

**Consensus:** before a new block is added, the entire network must agree it is valid. On WAX, this consensus takes just 1.5 seconds.

### No Single Owner

Blockchain is decentralized. No central server controls everything. Thousands of computers (nodes) maintain identical copies of the ledger. To defraud the system, an attacker would need to control more than half the network — impractical on established blockchains.

### Cannot Be Erased

Once a transaction is recorded and confirmed, it cannot be changed or removed. This is called immutability. For CryptoBingo, it means every draw, every prize, and every transaction is recorded forever.

### Everyone Can See

All transactions are public. Anyone can verify how many tickets were sold, how much was paid in prizes, and whether results were fair.

## Why Blockchain Matters for Gaming

Blockchain solves three fundamental problems of traditional online gaming:

**No hidden fees:** you know exactly how much you pay per ticket. The contract does not change the rules mid-game.

**Verifiable results:** every number drawn can be independently verified. The draw code is public.

**Automatic prizes:** when you win, payment is processed by the contract — no human intervention, no delays.

**Zero trust needed:** you don't need to trust the "house." The code is the guarantee.

## Blockchain vs Traditional Database

| Feature | Blockchain | Traditional Database |
|---------|------------|---------------------|
| Control | Decentralized (many nodes) | Centralized (one server) |
| Data changes | Immutable | Editable |
| Transparency | Public (anyone can see) | Private (operator only) |
| Availability | High (thousands of nodes) | Server-dependent |

## What Blockchain Is Not

**It is not just cryptocurrency.** Blockchain is infrastructure. Cryptocurrencies are just one application — like email is an application of the internet.

**It is not necessarily slow.** WAX processes blocks every 1.5 seconds with zero fees. Many people use blockchain without knowing it — WAX Cloud Wallet creates an account in 30 seconds with just email and password.

**It is not complicated to use.** The user interface looks like any website. The blockchain runs invisibly underneath.
