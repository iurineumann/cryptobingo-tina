## Security is the Foundation

At CryptoBingo, security is not a feature — it is the foundation. As an on-chain GameFi platform, every move is recorded on a public, auditable, and immutable blockchain. There is no "trust us" — there is "check for yourself."

## Fund Protection

**Auditable smart contract:** the CryptoBingo contract is on-chain on WAX. Anyone can read the code and verify rules — no black box.

**Connected wallet, not custodied:** your funds stay in your wallet (WAX Cloud Wallet or Anchor). CryptoBingo never custodies your tokens. You only authorize specific transactions (buy tickets, receive prizes).

**Auto-payout:** prizes are paid by the contract automatically. No withdrawal queue, no manual approval, no "under review." If you won, the contract pays.

## Draw Integrity

Every draw goes through three verification layers:

**1. Provably Fair:** before the draw, the hash of the future result is recorded on the blockchain. After the draw, you can verify that hash matches the result — ensuring no one altered the numbers afterward.

**2. Hybrid RNG:** the random number combines player commitment + server commitment + oracle RSA signature + nonce. No single party can predict or manipulate.

**3. Independent oracle:** the random number is generated by an external oracle (orng.wax) using RSA-SHA256. Not even CryptoBingo controls the result.

Read more in Provably Fair and How RNG Works.

## Player Privacy

**No mandatory KYC:** connect your wallet and play. No documents, no selfie required.

**Pseudonymity:** your identity on the blockchain is your WAX address — an alphanumeric hash. No personal data is stored on-chain.

**Minimal off-chain data:** only non-sensitive metadata (preferences, match history) in MongoDB with encryption at rest.

## Platform Protection

**Rate limiting:** API endpoints have request limits to prevent abuse.

**RBAC (Access Control):** administrative actions use granular permissions (admins, moderators), all logged.

**JWT with key rotation:** session tokens expire and auto-renew.

**Empty wallet cleanup:** the worker periodically removes accounts with no balance, freeing network resources.

## Honest Risks

**Contract risk:** although the code is public and auditable, no smart contract is 100% bug-free. We maintain constant evolution with testing.

**Human error:** you might send tokens to the wrong address or lose wallet access. Always verify addresses and guard your seed phrase.

**Market risk:** the value of WAX prizes can fluctuate. Dollar value is not fixed.

## Auditing and Transparency

All contract actions are public:
- Ticket purchases
- Draws and results
- Prize payments
- Game configurations
- Treasury movements

Anyone can monitor through block explorers like the WAX Block Explorer.

Security at CryptoBingo is not a promise — it is code. And code you can verify.
