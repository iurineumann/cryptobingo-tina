## 什么是质押？

质押是将你的 WAX 代币锁定以换取网络资源（CPU/NET）和奖励的过程。在 WAX 上，质押是交易的必要条件——你不支付费用，但需要资源。

## 为什么质押？

**为了游戏：** 没有最低质押，你的交易会因"CPU limit exceeded"而失败

**为了获得奖励：** 质押还能产生收益（APY 可变）

**为了治理：** 质押的 WAX 提供社区提案的投票权

## 如何质押

**通过WAX Cloud Wallet：**
1. 登录 wallet.wax.io
2. 前往设置 → 资源
3. 选择用于 CPU 和 NET 的 WAX 数量
4. 确认

**通过bloks.io：**
1. 在 wax.bloks.io 连接你的钱包
2. 前往钱包 → 资源
3. 将 WAX 质押到 CPU 和 NET
4. 选择要投票的区块生产者（可选，用于获得奖励）

## 需要质押多少？

CryptoBingo 最低建议：5 WAX 到 CPU，1 WAX 到 NET。

如果你频繁游戏或使用多张钱包，增加到 20 WAX 到 CPU。

## 解除质押

要撤销 WAX 的质押：
1. 前往钱包 → 资源
2. 点击"解除质押"
3. 等待 72 小时（WAX 解除质押期）
4. WAX 再次可用

## WAX治理

**投票：** 质押的 WAX = 投票权。你投票给维护网络的区块生产者（21 个席位）。

**提案：** WAX 社区提出变更——费用、协议升级、新功能。

**质押奖励：** 通过投票给区块生产者，你获得与你质押成比例的奖励。

## 风险

- **解除期：** 质押的 WAX 在解除时被锁定 72 小时
- **波动性：** APY 随 WAX 通胀率变化
- **削减：** WAX 风险最小（与其他 PoS 链不同）

质押是使用 WAX 的必要条件。从小额开始，随活动增加而增加。
