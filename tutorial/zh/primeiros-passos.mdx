## 第一次？开始吧！

欢迎来到 CryptoBingo — WAX 上首个链上 BINGO 平台。本指南用 7 个步骤带你从零到第一次获奖。

## 你需要什么

开始游戏前，请确保拥有：

1. **一个 WAX 钱包** — [WAX Cloud Wallet](https://wallet.wax.io)（通行密钥，推荐）或 [Anchor Wallet](https://www.anchor.link)（自托管）
2. **WAX 代币** — 用于购买门票（无 Gas 费；WAX 使用质押的 CPU/NET 资源）
3. **现代浏览器** — Chrome、Firefox 或 Brave

## 第一步：设置钱包

没有钱包？没问题。

- **WAX Cloud Wallet（30 秒）：** 访问 [wallet.wax.io](https://wallet.wax.io)，点击 "Create Account"，选择 12 字符用户名，使用 Face ID / Touch ID 作为通行密钥
- **Anchor Wallet（5 分钟）：** 从 [anchor.link](https://www.anchor.link) 下载，离线保存助记词，创建 WAX 账户

从 WAX Cloud Wallet 开始。当拥有更多代币后，可迁移到 Anchor 以获得完整的密钥控制权。

→ 完整指南：[如何创建您的 WAX 钱包](/zh/tutorials/criar-carteira-wax)

## 第二步：获取 WAX 代币

您需要 WAX 来购买门票。三种方式：

- **中心化交易所（需 KYC）：** Binance、KuCoin、Gate.io — 购买 USDT，兑换为 WAX，提现到您的钱包
- **Alcor Exchange（无需 KYC）：** 在 [alcor.exchange](https://alcor.exchange) 连接钱包，直接在 WAX 上兑换代币
- **测试网水龙头：** 在测试网上免费获取 WAX 进行练习

WAX 转账约 2 秒到账。

→ 完整指南：[如何购买 WAX](/zh/tutorials/comprar-wax)

## 第三步：连接到 CryptoBingo

1. 访问 [cryptobingo.bet](https://cryptobingo.bet)
2. 点击 "Connect Wallet"（右上角）
3. 在钱包中批准 — Cloud Wallet 弹出通行密钥，Anchor 显示二维码
4. 您的 WAX 用户名显示在页头 — 连接成功

## 第四步：选择游戏房间

在大厅中浏览房间。每个房间显示：

| 特性 | 说明 |
|---|---|
| **门票价格** | 入场费用（例如 10 WUF、50 WUF、100 WUF） |
| **游戏模式** | Standard、Fixed、First Full、Golden Ball、Multi Match |
| **奖池** | 可赢取的总 WAX 数量 |
| **玩家** | 当前参与者 |

**新手建议：** 从最便宜的 Standard 模式房间开始。

## 第五步：购买门票

1. 点击房间进入
2. 选择门票数量（1 到 10+）
3. 在钱包中确认交易
4. 等待约 2 秒（WAX 最终确认）
5. 您的 5x5 卡片显示在屏幕上

每张门票有 24 个数字 + 1 个自由格（数字 1–75）。组合由您的门票种子确定性生成 — 从一开始就是可验证公平的。

→ 完整指南：[如何购买门票](/zh/tutorials/comprar-tickets)

## 第六步：观看抽奖

房间满员或计时器到期后，抽奖开始：

- 数字逐个实时抽出
- 每次抽奖使用可验证公平的混合 RNG（玩家种子 + 服务器种子 + Oracle RSA-SHA256）
- 您匹配的数字会立即高亮显示
- 完成模式会触发中奖通知

**RTP 85%：** CryptoBingo 将 85% 的所有买入金额返还为奖品（15% GGR）。奖品计算：`支付价格 × base_multiplier × weight/100 × golden_multiplier`。

## 第七步：领取奖品

**奖品自动发放。** 无需申请提现。

1. 合约评估所有卡片与抽出的数字
2. 每张中奖卡片将 WAX 直接转入您的钱包
3. 屏幕上出现 "Winner!" 通知
4. 查看 "My Prizes" 获取完整历史记录

整个过程在抽奖结束后约 2 秒内完成。

## 常见问题解答

### 我无法连接钱包

请确保钱包扩展已安装并解锁。刷新页面重试。对于 Anchor，确认二维码扫描完成。如果按钮无响应，请清除浏览器缓存。

### 交易失败并显示 "CPU limit exceeded"

您需要向 CPU 质押更多 WAX。访问 [wax.bloks.io](https://wax.bloks.io)，连接钱包，进入 Wallet → Resources，向 CPU 质押更多 WAX。最低建议：5 WAX。

### 我没有收到奖品

1. 确认您完成了中奖模式（查看 "My History"）
2. 检查您的钱包余额 — 奖品可能已自动到账
3. 等待 WAX 最终确认（约 2 秒）
4. 在 [wax.bloks.io](https://wax.bloks.io) 上查找交易
5. 如果问题仍然存在，请提供您的 match ID 联系支持
