## 创建钱包之后

恭喜！现在我们来设置安全配置并连接到 CryptoBingo。

## WAX Cloud Wallet

1. 登录 [wallet.wax.io](https://wallet.wax.io)
2. 进入设置 → 安全
3. 启用**双因素认证**（Google Authenticator）
4. 验证你的邮箱
5. 连接到 CryptoBingo：点击"连接钱包"并授权

## Anchor Wallet

1. 打开 Anchor 并创建/导入你的钱包
2. 进入设置 → 安全
3. 设置强密码
4. 切勿分享你的助记词
5. 连接到 CryptoBingo：点击"连接钱包" → Anchor → 扫描二维码

## 安全检查清单

- [ ] 助记词**离线**保存（纸质，非数字形式）
- [ ] 已启用双因素认证（Cloud Wallet）或强密码（Anchor）
- [ ] 游戏钱包与主钱包分开（推荐）
- [ ] 浏览器中已验证 cryptobingo.bet URL
- [ ] 已删除可疑的浏览器扩展

## 测试连接

在 CryptoBingo 上，你的 WAX 用户名应显示在顶部。你的余额应可见。尝试购买一张彩票进行确认。
