## 独立验证

任何人都可以验证抽奖是公平的。你不需要相信我们——数据在区块链上。

## 验证什么

对于每次抽奖，你可以检查：

1. **抽奖前提交：** 服务器种子哈希是否在抽奖前记录？
2. **Oracle数字：** 随机数是否由 orng.wax oracle 签名？
3. **你的种子：** 你的彩票的个人种子是否正确？
4. **最终结果：** 组合哈希是否产生你看到的结果？

## 分步指南

**1. 找到比赛**

在网站上，前往"我的历史"并点击比赛 ID。复制交易 ID。

**2. 在区块浏览器上检查**

访问 [wax.bloks.io](https://wax.bloks.io) 并粘贴交易 ID。

**3. 检查提交**

查找 `startdraw` 或 `lockgame` 操作。服务器种子哈希在那里。

**4. 检查Oracle**

查找 `drawrand` 操作。查看由 oracle 签名的随机数。

**5. 检查结果**

查找 `claimwin` 操作。查看抽出的数字和支付的奖励。

## 工具

- **WAX区块浏览器：** wax.bloks.io — 交易、操作、原始数据
- **CryptoBingo验证器**（即将推出）：自动化验证的开源工具
- **验证脚本：** GitHub 上的代码，用于编程验证

## 提示

在抽奖开始前截图哈希值。抽奖后，进行比较。如果匹配，则抽奖未被操纵。
