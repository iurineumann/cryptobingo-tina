## WAX有资源，无费用

在 WAX 上，你不需要为每笔交易支付 Gas 费。取而代之的是，你需要资源：CPU、NET 和 RAM。

## CPU

处理时间。每笔交易消耗 CPU。你的限额基于你质押的 WAX 数量。

**CPU不足的症状：** 交易失败，提示"CPU limit exceeded"

**解决方法：** 在钱包或 [wax.bloks.io](https://wax.bloks.io) 上将更多 WAX 质押到 CPU

## NET

网络带宽。与 CPU 类似，需要质押 WAX。

**NET不足的症状：** 交易失败，提示"NET limit exceeded"

**解决方法：** 将 WAX 质押到 NET

## RAM

存储链上数据（余额、NFT 等）的内存。RAM 是购买的，不是质押的。

**RAM不足的症状：** 购买门票或创建 NFT 时提示"RAM limit exceeded"

**解决方法：** 在市场上（wax.bloks.io）或在交易所购买更多 RAM

## 需要质押多少？

休闲玩 CryptoBingo：
- CPU：质押 5-10 WAX
- NET：质押 1-2 WAX

如果同时使用多张钱包，按比例增加。

## 如何管理

在 [wax.bloks.io](https://wax.bloks.io)：
1. 连接你的钱包
2. 前往钱包 → 资源
3. 将 WAX 质押到 CPU 和 NET
4. 如有需要，购买 RAM

也可以直接在 WAX Cloud Wallet 中管理（设置 → 资源）。
