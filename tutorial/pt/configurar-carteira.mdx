## Depois de Criar a Carteira

Parabéns! Agora vamos configurar segurança e conectar ao CryptoBingo.

## WAX Cloud Wallet

1. Faça login em [wallet.wax.io](https://wallet.wax.io)
2. Vá em Settings → Security
3. Ative **2FA** (Google Authenticator)
4. Verifique seu email
5. Conecte ao CryptoBingo: clique "Connect Wallet" e autorize

## Anchor Wallet

1. Abra o Anchor e crie/importe sua wallet
2. Vá em Settings → Security
3. Defina uma senha forte
4. Nunca compartilhe sua seed phrase
5. Conecte ao CryptoBingo: clique "Connect Wallet" → Anchor → escaneie QR code

## Checklist de Segurança

- [ ] Seed phrase guardada OFFLINE (papel, não digital)
- [ ] 2FA ativado (Cloud Wallet) ou senha forte (Anchor)
- [ ] Carteira de jogo separada da principal (recomendado)
- [ ] URL cryptobingo.bet verificada no navegador
- [ ] Extensões de navegador suspeitas removidas

## Testando a Conexão

No CryptoBingo, seu nome de usuário WAX deve aparecer no topo. Seu saldo deve estar visível. Teste comprando um ticket para confirmar.
