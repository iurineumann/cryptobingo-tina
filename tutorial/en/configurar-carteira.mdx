## After Creating Your Wallet

Congratulations! Now let's set up security and connect to CryptoBingo.

## WAX Cloud Wallet

1. Log in at [wallet.wax.io](https://wallet.wax.io) (use your passkey)
2. Go to Settings → Security
3. Configure security (passkey, mnemonic phrase, 2FA)
4. Verify your email
5. Connect to CryptoBingo: click "Connect Wallet" and authorize

## Anchor Wallet

1. Open Anchor and create/import your wallet
2. Go to Settings → Security
3. Set a strong password
4. Never share your seed phrase
5. Connect to CryptoBingo: click "Connect Wallet" → Anchor → scan QR code

## Security Checklist

- [ ] Seed phrase stored OFFLINE (paper, not digital)
- [ ] Security configured: passkey + mnemonic + 2FA (Cloud Wallet) or strong password (Anchor)
- [ ] Gaming wallet separate from main wallet (recommended)
- [ ] cryptobingo.bet URL verified in browser
- [ ] Suspicious browser extensions removed

## Testing the Connection

On CryptoBingo, your WAX username should appear at the top. Your balance should be visible. Test by buying a ticket to confirm.
