## Before You Start

You need a WAX wallet to play CryptoBingo. Two options:

## Option 1: WAX Cloud Wallet (Recommended)

Simplest. Works like an email account.

1. Go to [wallet.wax.io](https://wallet.wax.io)
2. Click "Create Account"
3. Choose a username (12 characters, a-z and 1-5)
4. Set email and password
5. Done! Wallet created in 30 seconds

**Pros:** no seed phrase to manage, email recovery
**Cons:** custodial (WAX holds the key)

## Option 2: Anchor Wallet (Self-Custody)

For those who want full key control.

1. Download Anchor Wallet from [anchor.link](https://www.anchor.link)
2. Install and select "Create New Wallet"
3. Save the 12-word seed phrase **OFFLINE** (paper, pen, safe)
4. Create WAX account paying one-time fee (~10 WAX)
5. Set wallet password

**Pros:** full key control
**Cons:** you are responsible for the seed phrase

## Important Tip

Start with WAX Cloud Wallet. Play, learn, get comfortable. When you have more tokens or want more control, migrate to Anchor.

## Next Step

Now that your wallet is created, set it up and buy WAX to play.
